# youtube-downloader

PyQt-based GUI wrapper for yt-dlp.  
PyQt로 제작된 yt-dlp 기반 YouTube 다운로드 프로그램입니다.

## Features / 기능

- Download YouTube videos (mp4) / 유튜브 영상 다운로드 (mp4)
- Extract audio as mp3 / mp3 오디오 추출
- Simple and clean GUI / 간단하고 직관적인 GUI
- Playlist download disabled / 플레이리스트 다운로드 비활성화

## Requirements / 요구 사항

- Python 3.11+
- PyQt6
- yt-dlp
- FFmpeg
